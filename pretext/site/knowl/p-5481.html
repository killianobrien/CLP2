Finally, we show that \(n-\log n \gt 0\text{,}\) for all \(n\ge 1\text{.}\) Set \(f(x)=x-\log x\text{.}\) Then \(f(1)=1 \gt 0\) and

\begin{equation*} f'(x) = 1 -\frac{1}{x} \ge 0\qquad\text{for all }x\ge 1 \end{equation*}

So \(f(x)\) is (strictly) positive when \(x=1\) and is increasing for all \(x\ge 1\text{.}\) So \(f(x)\) is (strictly) positive for all \(x\ge 1\text{.}\)
