Solution 2.2.2.7.1

By definition, the average value is

\begin{gather*} \frac{1}{\pi}\int_{-\pi/2}^{\pi/2} \big( \sin(5x)+1 \big) \,\dee{x} \end{gather*}

We now observe that \(\sin(5x)\) is an odd function, and hence its integral over the symmetric interval \([-\frac\pi2,\frac\pi2]\) equals zero. So the average value of \(f(x)\) on this interval is \(1\text{:}\)

\begin{align*} \frac{1}{\pi}\int_{-\pi/2}^{\pi/2} \big( \sin(5x)+1 \big) \,\dee{x}&= \frac{1}{\pi}\int_{-\pi/2}^{\pi/2} \sin(5x) \,\dee{x}+ \frac{1}{\pi}\int_{-\pi/2}^{\pi/2}1 \,\dee{x}\\ &= \frac{1}{\pi}\int_{-\pi/2}^{\pi/2}1 \,\dee{x}=1 \end{align*}

Alternatively, using the fundamental theorem of calculus, the average equals:

\begin{align*} \amp\frac{1}{\pi}\bigg[\frac{-\cos (5x) }{5} +x \bigg]_{-\pi/2}^{\pi/2}\\ \amp\hskip0.1in= \frac{1}{\pi}\bigg\{\bigg[\frac{-\cos (5\pi/2) }{5} +\frac{\pi}{2}\bigg] -\bigg[ \frac{-\cos (-5\pi/2) }{5} + \frac{-\pi}{2} \bigg]\bigg\}\\ \amp\hskip0.1in= \frac{\pi }{\pi}=1 \end{align*}
