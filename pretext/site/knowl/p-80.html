  1. \(\sum\limits_{i=0}^n ar^i = a\frac{1-r^{n+1}}{1-r}\text{,}\) for all real numbers \(a\) and \(r\ne 1\) and all integers \(n\ge 0\text{.}\)
  2. \(\sum\limits_{i=1}^n 1 = n\text{,}\) for all integers \(n\ge 1\text{.}\)
  3. \(\sum\limits_{i=1}^n i = \frac{1}{2}n(n+1)\text{,}\) for all integers \(n\ge 1\text{.}\)
  4. \(\sum\limits_{i=1}^n i^2 = \frac{1}{6}n(n+1)(2n+1)\text{,}\) for all integers \(n\ge 1\text{.}\)
  5. \(\sum\limits_{i=1}^n i^3 = \Big[\frac{1}{2}n(n+1)\Big]^2\text{,}\) for all integers \(n\ge 1\text{.}\)
