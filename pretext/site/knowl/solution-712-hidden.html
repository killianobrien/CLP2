The Taylor Series for \(e^x\) is not alternating, so we'll use TheoremÂ 3.6.3 to bound the error in a partial-sum approximation. The error in the partial-sum approximation \(S_N\) is

\begin{equation*} E_N=\frac{f^{(N+1)}(c)}{(N+1)!}(x-a)^{N+1} \end{equation*}

for some \(c\) strictly between \(a\) and \(x\text{.}\) In our case, \(a=0\) and \(x=1\text{.}\) So, we want to find a value of \(N\) such that

\begin{equation*} \left|\frac{f^{(N+1)}(c)}{(N+1)!}\left(1-0\right)^{N+1}\right|=\frac{e^c}{(N+1)!} \lt 5\times 10^{-11} \end{equation*}

for all \(c\) in \((0,1)\text{.}\)

If \(c\) is between 0 and 1, then \(e^c\) is between 1 and \(e\text{.}\) However, since the purpose of this problem is to approximate \(e\) precisely, it doesn't make much sense to use \(e\) in our bound. Since \(e\) is less than 3, then \(e^c \lt 3\) for all \(c\) in \((0,1)\text{.}\) Now we can search for an appropriate value of \(N\text{.}\)

\(N\) \(\dfrac{3}{(N+1)!}\)
10 \(\displaystyle \frac{3}{11!}=\frac{1}{9^{10}}\approx 8\times 10^{-8}\)
11 \(\displaystyle \frac{3}{12!}\approx 6\times 10^{-9}\)
12 \(\displaystyle \frac{3}{13!}\approx 5\times 10^{-10}\)
13 \(\displaystyle \frac{3}{14!}\approx 3\times 10^{-11}\)

So, it suffices to use the partial sum \(S_{13}\text{.}\)
