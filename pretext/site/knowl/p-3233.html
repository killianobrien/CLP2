By Definition 2.1.1, the work done in moving the object from \(x = 1\) meters to \(x = 16\) meters by the force \(F(x)\) is

\begin{gather*} W = \int_1^{16} F(x)\dee{x} = \int_1^{16} \frac{a}{\sqrt{x}}\dee{x} = \Big[2a\sqrt{x}\,\Big]_{x=1}^{x=16} =6a \end{gather*}

To have \(W=18\text{,}\) we need \(a=3\text{.}\)
