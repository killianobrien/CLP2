The Taylor Series for \(\log(1-x)\) is not alternating, so we'll use Theorem 3.6.3 to bound the error in a partial-sum approximation. The error in the partial-sum approximation \(S_N\) is

\begin{equation*} E_N=\frac{f^{(N+1)}(c)}{(N+1)!}(x-a)^{N+1} \end{equation*}

for some \(c\) strictly between \(a\) and \(x\text{.}\) In our case, \(a=0\) and \(x=\dfrac1{10}\text{.}\) So, we want to find a value of \(N\) such that

\begin{equation*} \left|\frac{f^{(N+1)}(c)}{(N+1)!}\left(\frac{1}{10}\right)^{N+1}\right| \lt 5\times 10^{-11} \end{equation*}

for all \(c\) in \((0,\frac1{10})\text{.}\)

To find this \(N\text{,}\) we to know \(f^{(N+1)}(x)\text{.}\) Just like when we create a Taylor polynomial from scratch, we'll differentiate \(f(x)\) several times, and look for a pattern.

\begin{align*} f(x)&=\log(1-x)&f^{(6)}(x)&=\frac{-2(3)(4)(5)}{(1-x)^6}\\ f'(x)&=\frac{-1}{1-x}&f^{(7)}(x)&=\frac{-2(3)(4)(5)(6)}{(1-x)^7}\\ f''(x)&=\frac{-1}{(1-x)^2}&&\vdots\\ f'''(x)&=\frac{-2}{(1-x)^3}&f^{(N+1)}(x)&=\frac{-N!}{(1-x)^{N+1}}\\ f^{(4)}(x)&=\frac{-2(3)}{(1-x)^4}\\ f^{(5)}(x)&=\frac{-2(3)(4)}{(1-x)^5} \end{align*}

Now we want a reasonable bound on \(f^{(N+1)}(c)\text{,}\) when \(c\) is in \((0,\frac{1}{10})\text{.}\) Note that in this range, \(1-c \gt 0\text{.}\)

\begin{alignat*}{2} &&0 & \lt c \lt \frac{1}{10}\\ &\Rightarrow&\qquad \frac{9}{10} & \lt 1-c \lt 1\\ &\Rightarrow&\qquad \left(\frac{9}{10}\right)^{N+1} & \lt (1-c)^{N+1} \lt 1\\ &\Rightarrow&\qquad 1& \lt \frac{1}{(1-c)^{N+1}} \lt \left(\frac{10}{9}\right)^{N+1}\\ &\Rightarrow&\qquad N!& \lt \frac{N!}{(1-c)^{N+1}} \lt N!\left(\frac{10}{9}\right)^{N+1} \end{alignat*}

This bound provides us with a “worst-case scenario” error. We don't know exactly what \(c\) is, but we don't need to — the bound above holds for all \(c\) between 0 and \(\frac{1}{10}\text{.}\)

Now we're ready to choose an \(N\) that results in a sufficiently small error bound.

\begin{align*} \left|\frac{\textcolor{blue}{f^{(N+1)}(c)}}{(N+1)!}\left(\frac{1}{10}\right)^{N+1}\right|& \lt \frac{\textcolor{blue}{N!\left(\tfrac{10}{9}\right)^{N+1}}}{(N+1)!}\left(\frac{1}{10}\right)^{N+1}\\ \amp=\frac{1}{9^{N+1}\cdot (N+1)}\\ \text{So, we want:}\qquad \frac{1}{9^{N+1}\cdot (N+1)}& \lt 5\times 10^{-11} \end{align*}

To find an appropriate \(N\text{,}\) we test several values.

\(N\) \(\dfrac{1}{9^{N+1}\cdot (N+1)}\)
8 \(\displaystyle \frac{1}{9\cdot 9^{9}}=\frac{1}{9^{10}}\approx 3\times 10^{-10}\)
9 \(\displaystyle \frac{1}{10\cdot 9^{10}}\approx 3\times 10^{-11}\)

So, it suffices to use the partial sum \(S_9\text{.}\)
