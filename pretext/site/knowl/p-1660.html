The strategy for dealing with these integrals is similar to the strategy that we used to evaluate integrals of the form \(\int \sin^m x\cos^n x\dee{x}\) and again depends on the parity of the exponents \(n\) and \(m\text{.}\) It uses  2 

\begin{align*} \diff{}{x}\tan x &= \sec^2 x & \diff{}{x}\sec x &= \sec x\,\tan x & 1+\tan^2x &= \sec^2 x \end{align*}

We split the methods for integrating \(\int \tan^m x\sec^n x\dee{x}\) into 5 cases which we list below. These will become much more clear after an example (or two).

  1. When \(m\) is odd and any \(n\) — rewrite the integrand in terms of \(\sin x\) and \(\cos x\text{:}\)
    \begin{align*} \tan^m x\,\sec^n x\dee{x} &=\left(\frac{\sin x}{\cos x}\right)^m \left(\frac{1}{\cos x}\right)^n\dee{x}\\ &=\frac{\sin^{m-1}x}{\cos^{n+m}x}\ \sin x\dee{x} \end{align*}
    and then substitute \(u=\cos x\text{,}\) \(\dee{u} = -\sin x\dee{x}\text{,}\) \(\sin^2x = 1-\cos^2x=1-u^2\text{.}\) See Examples 1.8.11 and 1.8.12.
  2. Alternatively, if \(m\) is odd and \(n \geq 1\) move one factor of \(\sec x\,\tan x\) to the side so that you can see \(\sec x\,\tan x\dee{x}\) in the integral, and substitute \(u=\sec x\text{,}\) \(\dee{u}=\sec x\,\tan x\,\dee{x}\) and \(\tan^2 x = \sec^2 x-1=u^2-1\text{.}\) See Example 1.8.13.
  3. If \(n\) is even with \(n\ge 2\text{,}\) move one factor of \(\sec^2 x\) to the side so that you can see \(\sec^2 x\dee{x}\) in the integral, and substitute \(u=\tan x\text{,}\) \(\dee{u}=\sec^2 x\,\dee{x}\) and \(\sec^2 x = 1+\tan^2 x=1+u^2\text{.}\) See Example 1.8.14.
  4. When \(m\) is even and \(n=0\) — that is the integrand is just an even power of tangent — we can still use the \(u=\tan x\) substitution, after using \(\tan^2x = \sec^2 x - 1\) (possibly more than once) to create a \(\sec^2 x\text{.}\) See Example 1.8.16.
  5. This leaves the case \(n\) odd and \(m\) even. There are strategies like those above for treating this case. But they are more complicated and also involve more tricks (that basically have to be memorized). Examples using them are provided in the optional section entitled “Integrating \(\sec x\text{,}\) \(\csc x\text{,}\) \(\sec^3 x\) and \(\csc^3 x\)”, below. A more straight forward strategy uses another technique called “partial fractions”. We shall return to this strategy after we have learned about partial fractions. See Example 1.10.5 and 1.10.6 in Section 1.10.
