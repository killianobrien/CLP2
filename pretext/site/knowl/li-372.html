As we saw in the last example,
\begin{gather*} \diff{}{x}\log x = \frac{1}{x} \end{gather*}
and if we naively use this here, then we will obtain
\begin{align*} \int_{-2}^{-1} \frac{1}{x}\dee{x} &= \log(-1)-\log(-2) \end{align*}
which makes no sense since the logarithm is only defined for positive numbers  8 .
