Compute the indefinite integrals

\begin{align*} \int \sqrt{2x+1}\dee{x} && \text{and} && \int e^{3x-2}\dee{x} \end{align*}

Solution:

  • Starting with the first integral, we see that it is not too hard to spot the complicated argument. If we set \(u(x)=2x+1\) then the integrand is just \(\sqrt{u}\text{.}\)
  • Hence we substitute \(2x+1 \rightarrow u\) and \(\dee{x} \rightarrow \frac{1}{u'(x)}\dee{u}=\frac{1}{2}\dee{u}\text{:}\)
    \begin{align*} \int \sqrt{2x+1}\dee{x} &= \int \sqrt{u} \frac{1}{2} \dee{u}\\ &= \int u^{1/2} \frac{1}{2} \dee{u}\\ &= \left( \frac{2}{3}u^{3/2}\cdot\frac{1}{2} +C\right)\bigg|_{u=2x+1}\\ &= \frac{1}{3} (2x+1)^{3/2} + C \end{align*}
  • We can evaluate the second integral in much the same way. Set \(u(x)=3x-2\) and replace \(\dee{x}\) by \(\frac{1}{u'(x)}\dee{u} = \frac{1}{3}\dee{u}\text{:}\)
    \begin{align*} \int e^{3x-2}\dee{x} &= \int e^u \frac{1}{3}\dee{u}\\ &= \left( \frac{1}{3} e^u + C\right)\bigg|_{u=3x-2}\\ &= \frac{1}{3} e^{3x-2} +C \end{align*}
