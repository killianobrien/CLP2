Solution 2.4.7.30.1

(a) The given differential equation is separable and we could solve it accordingly. In fact we have already done so. If we rewrite the equation in the form

\begin{gather*} \diff{B}{t} = a\Big(B-\frac{m}{a}\Big) \end{gather*}

it is of the form covered by Theorem 2.4.4. So that theorem tells us that the solution is

\begin{gather*} B(t) = \left(B(0)-\frac{m}{a}\right) e^{at} + \frac{m}{a} \end{gather*}

In this problem we are told that \(a=0.02=\frac{1}{50}\text{,}\) so

\begin{gather*} B(t) = \left\{B(0)-50m\right\} e^{t/50} + 50m = \left\{30000-50m\right\} e^{t/50} + 50m \end{gather*}

(b) The solution of part (a) is independent of time if and only if \(30000-50m=0\text{.}\) So we need

\begin{gather*} m= \frac{30000}{50} = \$600 \end{gather*}
