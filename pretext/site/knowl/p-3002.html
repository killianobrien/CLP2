(b) We can cancel out the square root if we use a trig substitution. Substitute \(x=\sqrt{2}\tan y\text{,}\) \(\dee{x}=\sqrt{2}\sec^2 y\,\dee{y}\text{.}\)

\begin{align*} \int_0^{\sqrt{2}}\frac{1}{(2+x^2)^{3/2}}\,\dee{x} \amp=\sqrt{2}\int_0^{\pi/4}\frac{\sec^2 y}{(2+2\tan^2y)^{3/2}}\,\dee{y}\\ \amp=\frac{1}{2}\int_0^{\pi/4}\cos y\,\dee{y} =\frac{1}{2} \bigg[\sin y\bigg]_0^{\pi/4}\\ \amp=\frac{1}{2\sqrt{2}} \end{align*}
