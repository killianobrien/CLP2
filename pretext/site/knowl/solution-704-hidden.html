Recall that \(e^x = \displaystyle\sum\limits_{k=0}^\infty\frac{x^k}{k!}\text{.}\) So

\begin{gather*} \sum_{k=0}^\infty\frac{1}{\pi^k k!} =\Big[\sum_{k=0}^\infty\frac{x^k}{k!}\Big]_{x=1/\pi} =\Big[e^x\Big]_{x=1/\pi} =e^{1/\pi} \end{gather*}

This series differs from the given one only in that it starts with \(k=0\) while the given series starts with \(k=1\text{.}\) So

\begin{gather*} \sum_{k=1}^\infty\frac{1}{\pi^k k!} =\sum_{k=0}^\infty\frac{1}{\pi^k k!} -\underbrace{1}_{k=0} =e^{1/\pi}-1 \end{gather*}
